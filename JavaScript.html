JavaScript

if/else – Executa diferentes blocos de códigos dependendo da condição. if (condição){
// Código a ser executado se a condição for verdadeira
} else{
// Código a ser executado se a condição for falsa }

O if/else no JavaScript executa diferentes comandos dependendo de suas condições. O "if" executa um comando (bloco de código) se a condição for verdadeira, e o "else" executa outro comando se a condição for falsa.

Diferença entre let e const – Utilizados para criar variáveis e constantes. let nota = 7;
const PI = 3.14;

O "let" é utilizado para criar variáveis, essa variável fica armazenada dentro de um bloco e ela só será acessível dentro desse bloco, e há a possibilidade de reatribuir um valor novo a essa variável. Já o "const" é utilizado para criar constantes, variáveis que o valor atribuido não pode ser alterado, a tentativa de reatribuir outro valor resulta em um erro.
