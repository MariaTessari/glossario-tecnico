Bem Vindos ao Glossário Técnico!

Esse site foi desenvolvido com o intuito de auxiliar em meios técnicos da programação, podendo servir como meio de consulta e pesquisa para estruturas.

Esse glossário é divido em três seções: Tags HTML, explicando algumas tags responsáveis pela estrutura, suas funções e exemplos, Propriedades CSS, explicando algumas propriedades responsáveis por estilizar a estrutura, com as suas funções e exemplos também, e a seção JavaScript, que está em andamento, mas contém uma explicação breve sobre uma estrutura.

O HTML é utilizado para estruturar o conteúdo, o deixando de forma organizada. Ele usa tags para identificar elementos como os parágrafos, links, imagens, títulos e etc.

Propriedades CSS são responsáveis pela estilização, para controlar a aparência. Cada propriedade controla algo diferente que você queira mudar, como por exemplo para você mudar a largura, cor, fonte e etc.

O JavaScript é uma linguagem de programação, utilizada principalmente para tornar o conteúdo interativo e dinâmico, um exemplo da utilização é o accordion.
