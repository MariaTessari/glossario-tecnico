Tags HTML

<div> – Elemento de bloco genérico para agrupar outros elementos.
div class="container";

A tag "div" é utilizada principalmente para criar um bloco para agrupar outros elementos do HTML, como visto nos exemplos acima.

<header> – Cria cabeçalhos
<header>
<h1>Título Principal da Página</h1>
</header>

A tag "header" é utilizada principalmente para criar e definir cabeçalhos para uma seção ou página. Normalmente esse cabeçalho contém informações como título e informações de navegação.

<footer> – Cria rodapés
<footer>
<p> Meu Site - Todos os direitos reservados </p>
</footer>

A tag "footer" é utilizada principalmente para fazer rodapé, que geralmente é localizado na parte de baixo da página. O rodapé pode conter informações como por quem foi desenvolvido o site, contatos, copyright, entre outras.

<input> – Elemento de bloco genérico para agrupar outros elementos.
<input placeholder="Exemplo">

A tag "input" é como uma caixa de texto, para que o usuário possa digitar a informação pedida. Muito utilizada para fazer login, cadastrando informações como e-mail e senha. Confira o exemplo abaixo:

<section> – Define seções dentro de um documento HTML.
<section>
<h2>Clientes</h2>
</section>

A tag "section" agrupa os elementos dentro do código deixando a estrutura mais lógica. Diferentemente da tag "div", que agrupa elementos genéricos, a tag "section" agrupa elementos que tenham que tenham um propósito ou tema em comum.

<button> – Cria botões clicáveis.
<button>Clique Aqui</button>

A tag "button" é utilizado para criar botões interativos em páginas, que podem levar para outras páginas, ou para formulários de preenchimento, ou pode realizar a função de enviar um formulário ou informações.

<span> – Agrupa elementos inline geralmente para fins de estilização.
<p>Este é um texto com <span style="color: red;">uma palavra em vermelho</span>/p>

A tag "span" no HTML é utilizada para agrupar elementos para (em grande maioria), fins de estilização. Um exemplo é quando você quer deixar uma palavra ou parte do texto em vermelho, e o resto preto (ou da mesma cor definida).

<script src="accordion.js"></script>

<ol> – Cria uma lista ordenada.
<ol>
<li>Passo 1: Abrir a porta</li>
<li>Passo 2: Ligar a luz</li>
</ol>

A tag "ol" cria uma lista ordenada, isto é, onde cada item é numerado demonstrando sua posição na lista e tem uma ordem específica, se a ordem dos itens for mudada, o significado muda também.

<ul> – Cria uma lista não ordenada.
<ul>
<li>Batata</li>
<li>Manteiga</li>
</ul>

A tag "ul" é utilizada para criar listas não ordenadas, com itens que são exibidos e identificados com marcadores, indicando que a ordem desses itens não é importante, e que se essa ordem for mudada, não muda o significado da lista. Essa tag pode ser usada, por exemplo, para listar ingredientes, ou lista de compras. Essa tag pode acabar sendo confundida com a "ol", porém elas são diferentes.

<li> – Define itens dentro de uma lista
<li>Item 1</li>

A tag "li" serve para definir os itens de uma lista, podendo ser em uma lista ordenada (ol) ou uma lista não ordenada (ul). Ela identifica e representa cada item da lista, estruturando o conteúdo.

<link> – Fazer uma relação entre o documento atual e recursos externos
<link rel="stylesheet" href="tags.css">

A tag "link" é utilizada para estabelecer uma relação entre o documento atual e páginas e recursos externos, como, por exemplo, páginas do CSS. Geralmente é incluída na parte do "head" do código HTML. Também pode ser usado para importar ícones, e o que vai definir o seu uso vão ser os atributos dessa tag, atributos como: rel, type, etc.
